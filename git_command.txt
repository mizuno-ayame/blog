git add .   (= ステージング・・・一時保存)
git commit -m "コミットメッセージ"   (= コミット・・・ローカルブランチに反映)
git push origin main   (= プッシュ・・・リモートリポジトリに反映)

git branch : ローカルブランチを確認する
git branch -a : リモートブランチとローカルブランチを確認する
git branch ブランチ名 : 新しくブランチを作成する
git checkout ブランチ名 : 別のブランチに移動する
git merge ブランチ名 : ブランチの変更内容を反映させる

ブランチの状況
mainブランチ・・・本番環境
devブランチ・・・開発環境用
test1ブランチ・・・作業用（feature）

やりたいこと
1,test1ブランチの作業内容をdevブランチに反映させ、
2,devブランチからmainブランチに対して「プルリクエスト」をする
